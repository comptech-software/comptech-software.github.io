In this lesson, you will learn how to use the datetime module. You will also learn how to create and use variables in print functions.

In its most basic form, a print function takes some text in quotations and prints it to the console.

print("your_text_here")

Variables can also be put in the print function, even alongside text.

num_apples = 10
print("I have " + num_apples + " apples.")

You can print out the current time with the help of the datetime module.

from datetime import datetime
now = datetime.now()
print(now)

Here is a common mistake when printing variables:

num_apples = 10
print("num_apples")
# This will print out the text "num_apples", not the value of the variable

In this lesson, you will learn how to create and manipulate lists, use the random module, and use for loops.

To create a list, use square brackets and separate each item with a comma.

my_list = ["pen", "pineapple", "apple", "pen"]

You can manipulate lists with the append function, the insert function, and the remove function.

my_list.remove("pineapple")
my_list.insert(0, "apple")
my_list.append("pineapple")

print(my_list)
# ["apple", "pen", "apple", "pen", "pineapple"]

Using the random module can be very helpful in for loops to make it so that they run a random number of times.

import random

random_num = random.randint(1, 10)

for i in range(random_num):
    print("The loop has run" + i + "times")

In this lesson, you will learn how to set up and use the EV3 robot.

Setting up the EV3

  1. Install the ev3dev operating system onto a microSD card
    • For more info, see https://www.ev3dev.org/docs/getting-started/
  2. Insert the microSD card into the EV3 and power it on
  3. Pair your computer and the EV3 through Bluetooth (USB is another option, if you prefer)
    1. Enable Bluetooth on your computer and make sure that it is visible
    2. On the EV3: Wireless and Networks > Bluetooth > ensure "Powered" is checked > Start Scan > pair with your computer
      • If using a school-provided MacBook, the computer's name will be its serial number
  4. Install these VSCode extensions: "Python", "ev3dev-browser", "LEGO® MINDSTORMS® EV3 MicroPython"
  5. Connect to EV3 using the “EV3DEV DEVICE BROWSER” tab near the bottom-left of VSCode in the “Explorer” view

In this lesson, you will learn how to use the CrowPi's ultrasonic sensor.


Lesson

First, import the time and GPIO modules.

import RPi.GPIO as GPIO
import time

Then, switch the board mode and store the necessary pin values.

GPIO.setmode(GPIO.BCM)

GPIO_TRIGGER = 18
GPIO_ECHO = 24

You will then need to define a function that uses the sensor's input to return a distance value. Here is one such implementation:

# This particular function will return a distance in centimeters
def distance():
    # Set trigger to high
    GPIO.output(GPIO_TRIGGER, True)

    # Set trigger to low after 0.01 ms
    time.sleep(0.00001)
    GPIO.output(GPIO_TRIGGER, False)

    # Store time when object is first detected
    while GPIO.input(GPIO_ECHO) == 0:
        start_time = time.time()

    # Store time when object is last detected
    while GPIO.input(GPIO_ECHO) == 1:
        end_time = time.time()

    time_elapsed = stop_time - start_time

    # Multiply by the speed of sound (34300 cm/s)
    # Divide by 2 to account for the time it takes the waves to bounce back to the sensor
    distance = (time_elapsed * 34300) / 2

    return distance

In this lesson, you will learn how to use the CrowPi's sound sensor.


Lesson

To use the sound sensor, you must import its module and set its pin.

import RPi.GPIO as GPIO

SOUND_PIN = 24

Next, configure the sensor properly.

GPIO.setmode(GPIO.BCM)
GPIO.setup(SOUND_PIN, GPIO.IN, pull_up_down=GPIO.PUD_UP)

Here is an example of how you can use the sensor's True or False output:

while True:
    if GPIO.input(SOUND_PIN):
        print("Sound detected")

You'll notice that the sound sensor does not display volume-- it either detects sound or it doesn't. To adjust the sensitivity of the sound sensor, use the small blue potentiometer next to it. Turn the dial clockwise to lower the sensitivity and vice versa.

In this lesson, you will learn how to use the CrowPi's 8x8 LED dot matrix.


Lesson

To use the dot matrix, there are quite a few modules that must be imported first.

import re
import time
from luma.led_matrix.device import max7219
from luma.core.interface.serial import spi, noop
from luma.core.render import canvas
from luma.core.virtual import viewport
from luma.core.legacy import text, show_message
from luma.core.legacy.font import proportional, CP437_font, TINY_FONT, SINCLAIR_FONT, LCD_FONT

Next, define the following function to interface with the matrix:

def main(msg, scroll_delay):
    cascaded = 1
    block_orientation = 90
    rotate = 0

    serial = spi(port=0, device=1, gpio=noop())
    device = max7219(serial, cascaded=cascaded or 1, block_orientation=block_orientation, rotate=rotate or 0)

    show_message(device, msg, fill="white", font=proportional(CP437_font), scroll_delay=scroll_delay)

You can now call this function, inputting your own values for msg and scroll_delay, to display messages on the matrix.
