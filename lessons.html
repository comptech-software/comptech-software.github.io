In this lesson, you will learn how to use the datetime module. You will also learn how to create and use variables in print functions.

In its most basic form, a print function takes some text in quotations and prints it to the console.

print("your_text_here")

Variables can also be put in the print function, even alongside text.

num_apples = 10
print("I have " + num_apples + " apples.")

You can print out the current time with the help of the datetime module.

from datetime import datetime
now = datetime.now()
print(now)

Here is a common mistake when printing variables:

num_apples = 10
print("num_apples")
# This will print out the text "num_apples", not the value of the variable

In this lesson, you will learn how to create and manipulate lists, use the random module, and use for loops.

To create a list, use square brackets and separate each item with a comma.

my_list = ["pen", "pineapple", "apple", "pen"]

You can manipulate lists with the append function, the insert function, and the remove function.

my_list.remove("pineapple")
my_list.insert(0, "apple")
my_list.append("pineapple")

print(my_list)
# ["apple", "pen", "apple", "pen", "pineapple"]

Using the random module can be very helpful in for loops to make it so that they run a random number of times.

import random

random_num = random.randint(1, 10)

for i in range(random_num):
    print("The loop has run" + i + "times")
